# Android-arduino-bluetooth

Coche controlado por bluetooth y aplicacion android

## Requisitos

* 1 Arduino nano
* 1 Puente H L298n
* 1 Bluetooth HC-06
* [bluetooth RC controller](https://play.google.com/store/apps/details?id=braulio.calle.bluetoothRCcontroller&hl=es-419)
